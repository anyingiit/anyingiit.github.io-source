# 安装fancybox插件

### 切换到lib目录

```shell
cd nex/source/lib
```

### 下载插件

```shell
git clone https://github.com/theme-next/theme-next-fancybox3 fancybox
```

### 生效插件

更改next/_config.yml文件

```shell
fancybox: true
```
